# ScholarMatch — Scholarly Paper Recommendation System

ScholarMatch is a machine learning application that helps students and researchers find similar academic papers quickly.

The system uses paper titles, abstracts and topics from the OpenAlex dataset. It converts the text into numerical vectors using TF-IDF and compares papers using cosine similarity.

Users can search for a paper, select it and receive a ranked list of related research papers with details such as authors, topics, publication year, citation count and similarity score.
